JAVA DSA & LEETCODE PROJECTS
===============================

PROJECT OVERVIEW
----------------
This repository contains multiple Java programs based on
Data Structures and Algorithms (DSA).
All problems are taken from LeetCode and solved using
clean, optimized, and interview-ready Java code.

This project is created to:
- Practice core DSA concepts
- Prepare for coding interviews
- Demonstrate problem-solving skills in Java

--------------------------------
PROJECTS INCLUDED 
--------------------------------

*1) LONGEST PALINDROMIC SUBSTRING
--------------------------------
• Finds the longest palindrome in a given string  
• Uses Expand Around Center technique  
• Covers string manipulation and logic building  
• Time Complexity: O(n²)

Skills Used:
- Strings
- Two pointers
- Optimization logic

--------------------------------

*2) LINKED LIST PROBLEMS
--------------------------------
Problems Covered:
• Linked List Cycle (Detect cycle)
• Linked List Cycle II (Find cycle start)
• Intersection of Two Linked Lists

Concepts Used:
- Slow & Fast Pointer (Floyd’s Algorithm)
- Reference comparison
- O(1) extra space solution

Skills Used:
- Linked List
- Pointers
- Memory efficient logic

--------------------------------

*3) HAPPY NUMBER
--------------------------------
• Checks whether a number is a Happy Number  
• Uses cycle detection logic  
• Avoids infinite loop using fast & slow pointer  

Skills Used:
- Mathematics
- Loop control
- Optimization

--------------------------------

*4) PASCAL’S TRIANGLE
--------------------------------
• Generates Pascal’s Triangle up to given rows  
• Uses dynamic list creation  
• Demonstrates pattern and array logic  

Skills Used:
- Arrays
- Lists
- Nested loops

--------------------------------

*5) ACCOUNTS MERGE
--------------------------------
• Merges multiple user accounts based on common emails  
• Uses Union-Find (Disjoint Set) algorithm  
• Emails are sorted for clean output  

Skills Used:
- HashMap
- TreeSet
- Union Find
- Real-world data merging logic

--------------------------------
TECHNOLOGIES USED
--------------------------------
• Java  
• Collections Framework  
• Data Structures & Algorithms  
• LeetCode problem-solving approach.
